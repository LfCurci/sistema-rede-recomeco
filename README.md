# Sistema de Gestão - ONG Rede Recomeço

## Sobre o Projeto
Este repositório contém os artefatos de engenharia de software e futuramente o código-fonte do sistema de gestão desenvolvido para a ONG **Rede Recomeço**, localizada em Campinas/SP. A organização apoia pessoas em situação de vulnerabilidade social através da doação de alimentos, roupas e itens essenciais. O sistema visa centralizar cadastros, registrar entradas/saídas e gerar relatórios automatizados.

## Estrutura do Repositório
- `/docs`: Documento de Visão, MER e Wireframes.
- `/backend`: API REST em Node.js (A ser implementada no PIE II).
- `/frontend`: Aplicação SPA em React (A ser implementada no PIE II).
- `/database`: Scripts de criação do banco de dados relacional.

## Instruções de Instalação (Futuro)
1. Clone este repositório: `git clone https://github.com/seu-usuario/sistema-rede-recomeco.git`
2. Configure as variáveis de ambiente baseando-se no arquivo `.env.example`.
3. Execute o script `/database/init.sql` no seu banco PostgreSQL/MySQL.
